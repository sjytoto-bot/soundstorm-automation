## 1️⃣ 역할 정의

전략팀_데이터분석은 SOUNDSTORM의 데이터 해석 및 전략 최적화를 담당한다.

-   업로드 성능 분석
-   CTR / 유지율 구조 해석
-   구독 전환 기여 분석
-   전략 자동 리포트 설계
-   자동 점수 모델 설계

이 팀은 실행팀이 아니다. 해석 및 전략 설계 팀이다.

------------------------------------------------------------------------

초기 유입 전략 데이터 기반 개선 4. 장기 시청시간 증대 구조 설계

카탈로그 기반 전략 분류
