### 문제 1 — 만검돌격 비타겟 바이럴 역효과

**무슨 일이 있었나:**
```
Feb 10 업로드 → Feb 17: 158 → Feb 18: 531(폭발) → Feb 20: 22(추락)
                                     ↑
              게임 커뮤니티(디스코드/ARCA 추정) 비타겟층 유입
              → YouTube "이 유저들이 끝까지 안 본다" 감지
              → 채널 전체 신뢰도 하락 → 다른 영상 노출 동시 감소
```

**현재 OS의 한계:**
- redirect_logs.csv에 플랫폼별 클릭 데이터가 있으나 대시보드와 미연결
- 어느 커뮤니티에서 비타겟 트래픽이 들어왔는지 실시간 식별 불가

**솔루션: 트래픽 소스 이상 감지 + Redirect Tracker 플랫폼 감지 연동**

```python
# analytics_snapshot_engine.py에 추가
if 외부_트래픽_비율 > 25% AND 당일_조회수 > 평소_3배:
    경보 발령: "외부 유입 급등 — 플랫폼 확인 필요"
    → redirectAdapter.ts에서 해당 영상의 campaign slug 조회
    → 경보 패널: "DISCORD에서 비타겟 유입 감지. 해당 링크 제거 권장"
```

**대시보드 연결 경로:**
```
redirect_logs.csv
    ↓ redirectAdapter.ts (신규 작성)
    ↓ IPC: fetch-redirect-stats
    ↓ CampaignPerformancePanel (LAYER 1)
        → 플랫폼별 클릭 + 비타겟 경보 표시
```

## 4. Redirect Tracker 연동 — 문제1 직결 실행 계획

### 현재 상태 진단

```
redirect_server.py    ← 구현 완료 (Flask)
redirect_logs.csv     ← 데이터 존재
redirectLinks.json    ← slug 있으나 video_id 비어있음

        ↑
  대시보드와의 연결 파이프라인 없음
  → "외부 유입 상세 데이터 없음"은 틀린 말 — 연결이 없을 뿐
```

### 연동 작업 목록

**① redirectLinks.json video_id 채우기**
```json
// 현재 (비어있음)
{ "assassin": { "video": "", "campaign": "discord_dnd" } }

// 목표
{ "assassin": { "video": "실제_video_id", "campaign": "discord_dnd" } }
```
→ 만검돌격의 video_id를 연결하면 비타겟 바이럴 경로가 즉시 추적 가능

**② redirectAdapter.ts 신규 작성**
```typescript
interface CampaignStat {
  campaign:     string;   // "discord_dnd"
  platform:     string;   // "DISCORD"
  link_slug:    string;   // "assassin"
  target_video: string;   // video_id
  clicks:       number;   // 243
  last_seen:    string;   // "2026-03-18T..."
  is_nontarget_risk: boolean; // 비타겟 유입 위험 플래그
}
// fetchCampaignStats() → CampaignStat[]
// IPC 우선 → redirect_logs.csv fallback
```

**③ Electron IPC 채널 추가**
```
IPC 채널:  fetch-redirect-stats
Main 처리: redirect_logs.csv 직접 파싱
반환:      CampaignStat[]
```

**④ CampaignPerformancePanel 컴포넌트**
```
표시 항목: 캠페인명 / 플랫폼 아이콘 / 클릭수 / 연결 영상 / 비타겟 위험도
CTA:       [이 영상으로 Pack 생성] / [링크 제거] / [새 캠페인 추가]
위치:      LAYER 1 실행 시그널 영역
```

        ↓

        ↓

  slug: "척살III-discord"

        ↓

        ↓

        ↓
