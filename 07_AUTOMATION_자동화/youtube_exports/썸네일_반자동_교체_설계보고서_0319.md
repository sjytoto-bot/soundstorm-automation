## 3. 반자동 워크플로우

```
[Step 1] 참조 썸네일 분석 (자동 — 30초)
  thumbnail_analyzer.py → 척살II/군주 URL 분석
  → style_tags, 색상 팔레트, 구도 정보 추출

[Step 2] Midjourney 프롬프트 생성 (자동 — 10초)
  prompt_generator.py --theme "흑검" --style dark,red_dominant,minimal
  → "dark warrior silhouette, black blade, red cinematic lighting,
     Korean battle scene, dark void background, minimal composition
     --ar 16:9 --v 6 --style raw --q 2"

[Step 3] 이미지 생성 (반자동 — Midjourney에서 2~3분)
  → Midjourney Discord에 프롬프트 붙여넣기
  → 생성된 이미지 4장 중 선택 → 다운로드
  → thumbnail_intelligence/uploads/ 에 저장

[Step 4] 텍스트 합성 (자동 — 10초)
  template_engine.py → 배경 이미지 + 텍스트 오버레이 합성
  → 척살II 텍스트 스타일 적용 (위치/폰트/크기 자동 계산)
  → output/ 에 1280×720 PNG 저장

[Step 5] 승인 후 업로드 (수동 — 2분)
  → output/ 파일 검토
  → YouTube Studio → 썸네일 교체
```

**자동화율: 5단계 중 3단계 자동 + 1단계 반자동(MJ 클릭) + 1단계 수동(업로드)**

### 핵심 함수 설계

```python
def run_pipeline(video_id, title, style_tags):
    """
    단일 영상 썸네일 교체 파이프라인

    Returns:
        output_path (Path) — 합성된 썸네일 파일 경로
    """
    _THIS_DIR = Path(__file__).parent

    # Step 1: 참조 썸네일 분석
    print(f"\n[1/4] 참조 스타일 분석 중...")
    ref_styles = analyze_reference_thumbnails(REFERENCE_VIDEOS)

    # Step 2: 프롬프트 생성
    print(f"[2/4] Midjourney 프롬프트 생성 중...")
    from prompt_generator import generate_prompt
    result  = generate_prompt(theme=title, style_tags=style_tags)
    prompt  = result["mj_prompt"]

    # 클립보드 복사 (macOS)
    subprocess.run(["pbcopy"], input=prompt.encode())
    print(f"\n  ✅ 프롬프트가 클립보드에 복사됐습니다:")
    print(f"  {prompt}\n")

    # Step 3: 이미지 대기 (uploads/ 감시)
    print(f"[3/4] Midjourney에서 이미지를 생성 후")
    print(f"      {_THIS_DIR}/uploads/ 에 {video_id}.png 으로 저장하세요.")
    upload_path = _THIS_DIR / "uploads" / f"{video_id}.png"
    _wait_for_file(upload_path)

    # Step 4: 텍스트 합성
    print(f"[4/4] 썸네일 합성 중...")
    from template_engine import render_thumbnail
    output_path = render_thumbnail(
        bg_path=str(upload_path),
        copy_text=title,
        output_name=f"thumb_{video_id}.jpg",
    )

    print(f"\n  ✅ 썸네일 생성 완료: {output_path}")
    print(f"  YouTube Studio에서 수동 업로드 필요")
    print(f"  https://studio.youtube.com/video/{video_id}/edit\n")

    return output_path


def _wait_for_file(path, timeout=300):
    """파일이 생길 때까지 대기 (최대 5분)"""
    start = time.time()
    while not path.exists():
        if time.time() - start > timeout:
            raise TimeoutError(f"파일 대기 시간 초과: {path}")
        print(f"  ⏳ 대기 중... ({path.name})", end="\r")
        time.sleep(3)
    print(f"\n  ✅ 파일 감지됨: {path.name}")
```
