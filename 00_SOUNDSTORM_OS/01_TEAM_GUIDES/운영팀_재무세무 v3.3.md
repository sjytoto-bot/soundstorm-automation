# 2️⃣ 내부 정책 레이어

## 2.1 재무세무 현재 상태 요약 — 기준점 (원문 보존)

■ 해결 완료

- 2025년 사업 총수입 약 2,100만원 확정
    
- 단순경비율(64.1%) 적용 가능 구간 확인
    
- 예상 사업소득 약 760만원 계산
    
- 지역가입자 건보료 월 6~8만원 예상
    
- 미국주식 양도소득 건보료 영향 거의 없음 정리
    
- 배당 20만원 영향 미미 확인
    
- 사하라자드 500만원 매출/외주비 구조 확정
    
- 개인→사업 통장 이체는 매출 아님 정리
    

■ 미해결

- 과거 5년 무신고 처리 전략 확정
    
- 김정준 외주 반복 발생 시 원천징수 처리 여부
    
- 사업·개인 자금 완전 분리 정착
    
- 종합소득세 신고 방식 최종 확정
    

■ 진행 중

- 2025년 간편장부 기준 소득 확정
    
- 건보료 영향 분석
    
- 계좌·카드 혼합 내역 분리 기준 정립
    

■ 다음 단계

1. 2025년 종합소득세 신고 준비 (단순경비율 적용)
    
2. 예상 세액 계산
