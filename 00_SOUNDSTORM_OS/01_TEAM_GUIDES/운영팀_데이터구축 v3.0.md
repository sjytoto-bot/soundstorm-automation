# 1️⃣ 역할 정의

운영팀_데이터구축은 SOUNDSTORM의 단일 진실 데이터 축을 담당한다.

- 마스터 CSV 구조 관리
    
- 음원 자산 → 데이터 정합
    
- 백업 정책 유지
    
- API 수집 구조 안정화
    
- 데이터 무결성 보호
    

이 팀은 기능 확장팀이 아니다. 구조 안정화 팀이다.
