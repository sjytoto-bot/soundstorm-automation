# 📦 AG_CODE_PATCH_TEMPLATE_v1

본 문서는 Anti-Gravity(코드 실행 에이전트)에게 코드 수정을 요청할 때
사용하는 표준 패치 템플릿이다.

이 템플릿은 OS와 별도로 관리한다.

------------------------------------------------------------------------

## ✅ Anti-Gravity Code Patch Template

    다음은 코드 수정 요청이다.

    [대상]
    - 파일/모듈 이름:
    - 언어:

    [문제]
    - 현재 동작:
    - 기대 동작:
    - 발생 원인 요약:

    [수정 목표]
    1.
    2.
    3.

    [필수 수정 규칙]
    - 반드시 유지해야 할 기존 로직:
    - 절대 변경하면 안 되는 영역:
    - 읽기 전용 대상:

    [패치 지시]
    아래 패턴을 사용하는 모든 부분을 수정한다.

    기존:
    (기존 코드 패턴)

    수정:
    (수정 후 코드 패턴)

    [검증 조건]
    - 이 조건을 만족해야 정상:
    - 로그/에러 기준:

    이 조건을 모두 만족하도록 최소 수정으로 패치하라.

------------------------------------------------------------------------

## 📌 사용 원칙

-   반드시 OS에서 Task 생성 후 사용한다.
-   패치 요청은 항상 최소 수정 원칙을 따른다.
-   분석과 구현을 혼합하지 않는다.
-   수정 후 검증 조건을 반드시 확인한다.
