# 5️⃣ Execution Instruction Standard v1.0

    [전달 지시]

    첨부 파일: snapshot_engine.py (Claude 생성, 테스트 통과)

    역할: 코드 수정 없이 실행만 할 것.

    1. SPEC_VERSION 일치 여부 확인
    2. 경로 존재 여부 확인

    실행 명령:
    python3 snapshot_engine.py draft  <팀명>
    python3 snapshot_engine.py approve <팀명>

    경로 기준:
    팀가이드: SOUNDSTORM/00_SOUNDSTORM_OS/01_TEAM_GUIDES/

    Archive:   07_AUTOMATION_자동화/ARCHIVE/

    코드 수정 금지.
    경로 다르면 수정하지 말고 보고 후 대기.
    테스트 실패 시 승인 금지.

# 6️⃣ Code Review Separation Rule

## 6.1 GPT 코드 검토 한정 규칙

GPT는 코드에 대해 다음만 수행할 수 있다: - 논리 오류 지적 - 보안 리스크
지적 - Spec 위반 여부 검토 - 엣지 케이스 제안

GPT는 다음을 수행할 수 없다: - 전체 코드 재작성 - 함수 단위 구현 대체 -
구현 로직 변경 - 최종 코드 블록 제공

------------------------------------------------------------------------

## 6.2 구현 책임 고정

최종 실행 코드의 구현 책임은 Claude Code / Codex에게만 있다.

GPT 제안 사항은 반드시 Claude가 재구현한 코드만 승인 대상이 된다.

------------------------------------------------------------------------

## 6.3 위반 시 처리

-   GPT가 코드 구현을 수행한 경우 → 폐기
-   구현 책임자 불명확한 코드 → 승인 금지
